# Algorithm Visualizer

A Python-based desktop application that visualizes popular searching, sorting, graph, and tree algorithms to help users understand how they work through interactive animations.

## Features

- Sorting Algorithms
  - Bubble Sort
  - Selection Sort
  - Merge Sort
  - Quick Sort

- Searching Algorithms
  - Linear Search
  - Binary Search

- Graph Algorithms
  - Breadth First Search (BFS)
  - Depth First Search (DFS)
  - Dijkstra's Algorithm

- Tree Traversal
  - Binary Tree Traversal

- Interactive GUI for visualization

## Technologies Used

- Python
- Tkinter
- Matplotlib
